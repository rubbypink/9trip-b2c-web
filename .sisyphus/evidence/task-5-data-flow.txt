Task 5: Data Flow Fix — childPrices merge + tier limit increase
================================================================

## Bug
`extractActivityPage()` in browser-automation.mjs returns `childPrices` (per-tier
child prices) and `childPricing` (from clickQuantityButtons), but
`scrapeActivityFromUrl()` in activityScraper.mjs NEVER merged them into
`pricing.tiers[]` or `pricing.childPrice/infantPrice/seniorPrice`.

## Fix 1: activityScraper.mjs — Destructure + merge childPrices/childPricing

### Change A: Destructure childPrices and childPricing from extractResult
Line 435 (was):
  const { bodyText, bodyHtml, title, url: rawUrl, clicked, errors } = extractResult;
Now:
  const { bodyText, bodyHtml, title, url: rawUrl, clicked, errors, childPrices, childPricing } = extractResult;

### Change B: Merge childPrices into pricing.tiers after scrapeActivityFromText()
After `scrapeResult.data._strategy = 'agent-browser';` (was line 457), added:

  // Merge per-tier child prices from browser extraction (extractChildPricesPerTier)
  if (childPrices && Object.keys(childPrices).length > 0) {
    for (const [tierIndex, priceData] of Object.entries(childPrices)) {
      const idx = parseInt(tierIndex);
      if (idx < scrapeResult.data.pricing.tiers.length && priceData.childPrice) {
        scrapeResult.data.pricing.tiers[idx].childPrice = priceData.childPrice;
      }
    }
  }

  // Merge childPricing from clickQuantityButtons if not already populated
  if (childPricing) {
    if (childPricing.childPrice && !scrapeResult.data.pricing.childPrice) {
      scrapeResult.data.pricing.childPrice = childPricing.childPrice;
    }
    if (childPricing.infantPrice && !scrapeResult.data.pricing.infantPrice) {
      scrapeResult.data.pricing.infantPrice = childPricing.infantPrice;
    }
    if (childPricing.seniorPrice && !scrapeResult.data.pricing.seniorPrice) {
      scrapeResult.data.pricing.seniorPrice = childPricing.seniorPrice;
    }
  }
